Pivotal Extension Framework (PXF)

About this Guide

This guide provides information for system administrators and application developers responsible for installing and customizing HAWQ using Pivotal Extension Framework (PXF).

This guide provides information and instructions for installing PXF, and using the APIs to extend HAWQ functionality.

This guide assumes knowledge of Linux/UNIX system administration, database management systems, REST APIs, HIVE, and HDFS.

This guide includes the following information:

  • Installing the PXF JAR file: Describes the pre-requisites and pre-installation checks for installing PXF. It also describes how you can install PXF. Describes how you can ensure security in your PXF installation.
  • Accessing HDFS Data with PXF: Reference information and examples showing how you can access HDFS File Data.
  • Accessing HIVE Data with PXF: Reference information and examples showing how you can access HIVE File Data.
  • Accessing HBASE Data with PXF: Reference information and examples showing how you can access HBASE File Data.
  • Accessing GemFire Data with PXF: Reference information and examples showing how you can access GemFire Data.
  • Pivotal Extension Framework External Table and API Reference: Describes how to create an external table with the PXF protocol and describes the PXF Java Classes and Formats.
